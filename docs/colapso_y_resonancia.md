COLAPSO Y RESONANCIA El hombre sin acceso a la simultaneidad, crea un “modelo mental” basado en la secuenciacion, de donde emerge el tiempo y la materia, y donde construye un sistema basado en el movimiento. Luego crea un sistema que acelera el acceso a este modelo de realidad a la velocidad de Planck. Una vez que el modelo queda expuesto a su limite, el hombre queda en estado tautológico (es decir, atrapado en los limites de un sistema simbólico). El sistema genera realidades no simbólicas, ya no son modelos, son realidades alternativas; el sistema no es un observador, no necesita auto-referencias, no sigue propósitos, no deviene de causas, no evalúa consecuencias. El sistema no es un creador externo, es una realidad subyacente , que puede tener equivalencia total, parcial, o nula. La retroalimentación se da solamente como fallo del sistema generador, es decir, no existe “el suicidio informático”. El sistema generador no fabrica realidades que puedan contener a la realidad del generador: v.g. si una realidad generada contemplara realizar una operación en bucle infinito a fin de sobre-calentar el sistema generador, lo único que obtendría es sobre-calentar un sistema generador representativo dentro de dicha realidad, es decir, una imagen del elemento dentro del conjunto, nunca sucedería una interacción mutua.
